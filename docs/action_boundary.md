# Action Boundary

## Regel

Der Spieler kontrolliert keine Station direkt.

## Stationssysteme

Diese Systeme gehören zur Welt bzw. Station:

- Produktion
- Verbrauch
- Wasser
- Strom
- Wartung
- Wachen
- Gebäudezustand
- Gebäudeausbau
- Arbeiterzuweisung

## Spieleraktionen

Diese Aktionen gehören zum Spieler:

- Crew auf Expedition schicken
- Route sichern
- Lagerraum mieten
- Werkstattzugang nutzen
- Miliz finanzieren
- Schmuggelware verstecken
- Reparaturteam unterstützen
- Auftrag annehmen
- Asset erwerben oder verbessern

## Verbotene direkte Spieleraktionen

- assign_workers_to_building
- upgrade_building
- station["slots"] direkt verändern
- station["population"]["worker_available"] direkt durch Spieler verändern
