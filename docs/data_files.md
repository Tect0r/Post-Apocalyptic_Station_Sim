# Data Files

## `data/world.json`

Defines the shared world:

- stations
- routes
- factions
- station markets
- route control

## `data/player_actions.json`

Defines free player actions:

- action type
- target type
- duration
- cost
- effects

## `data/contracts.json`

Defines available contracts:

- title
- issuer
- target
- action type
- cost
- reward
- world effects

## `data/player_assets.json`

Defines asset types:

- max level
- effects by level
- upgrade costs
- repair costs

## `data/market_items.json`

Defines tradeable items:

- label
- category
- buy/sell base price
- tradeable flag

## `data/world_event_rules.json`

Defines event triggers and effects.

## `data/pvp_rules.json`

Defines indirect PvP limits:

- cooldowns
- maximum effect
- detection chance
- reputation cost
